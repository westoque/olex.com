# EmailObject

The EmailObject.

| Field          | Type        | Required   | Description
|----------------|-------------|------------|------------------------------------------
| emailAddress   | string      | true       | The email address in valid format.
| isUnsubscribed | boolean     |            | If the email is unsubscribed.

## Example

```
{
  emailAddress: "foo@bar.com",
  isUnsubscribed: false
}
```
